# 🔧 Scripts

This directory contains utility scripts and helper files for the project.

## 📁 Files

### Shell Scripts (`.sh`)
- `setup-env.sh` - Environment setup script
- `verify-setup.sh` - Verify project setup
- `run.sh` - Run project script
- `fix-db-permissions.sh` - Fix database permissions
- `fix-permissions-quick.sh` - Quick permission fix

### JavaScript Utilities (`.js`)
- `check-env.js` - Check environment variables
- `verify-env.js` - Verify environment configuration

## 🚀 Usage

Most scripts can be run directly:
```bash
./scripts/setup-env.sh
./scripts/check-env.js
```

Or with appropriate permissions:
```bash
chmod +x scripts/*.sh
./scripts/setup-env.sh
```

---

**Note:** Make sure scripts have execute permissions before running.
